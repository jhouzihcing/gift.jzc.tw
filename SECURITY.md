# Security Policy

## Supported Versions

Currently, only the latest version of this project is supported. We recommend always keeping your local copy up to date.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly by using the **[GitHub Security Advisories](https://github.com/houzihcing/gift.jzc.tw/security/advisories/new)** feature.

We process all reports as quickly as possible and will keep you updated on our progress. Thank you for helping keep this project secure!
